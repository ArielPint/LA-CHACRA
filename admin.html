Los usuarios se guardan en users.json del repositorio GitHub. Al guardar aquí, todos los equipos ven los cambios en su próximo login.

Cómo obtener el token:
1. Ve a GitHub → Settings → Fine-grained tokens
2. Repository access: ArielPint/LA-CHACRA
3. Permissions → Contents: Read and write
4. Genera el token y pégalo aquí. Solo necesitas hacerlo una vez en este equipo.

Roles predefinidos

Permisos por defecto al asignar un rol

🔑 Administrador

  • Acceso a todas las páginas
  • Todas las pestañas
  • Puede editar
  • Accede al panel admin

⚙️ Operador

  • Acceso a todas las páginas
  • Todas las pestañas
  • Puede editar
  • Sin acceso al panel admin

👁️ Solo Lectura

  • Acceso a todas las páginas
  • Todas las pestañas
  • Solo visualización
  • Sin acceso al panel admin
