Credenciales GeoVictoria

Las credenciales se obtienen en la plataforma GeoVictoria:
Configuraciones Empresa → Acceso API → copiar "Clave Api" y "Secreto".

El host que te indicó tu jefe de proyecto GeoVictoria.

Nota de seguridad: Las credenciales se guardan en el localStorage del navegador y nunca se envían a servidores externos (sólo se usan para llamar directamente a la API de GeoVictoria).

CORS: Si ves errores de CORS, GeoVictoria debe autorizar el dominio de La Chacra en su configuración. Contacta a tu jefe de proyecto GeoVictoria.
